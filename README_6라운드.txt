배포 방법
1. 이 ZIP의 index.html을 기존 GitHub 저장소 최상단에 덮어씁니다.
2. 커밋이 완료되면 GitHub Pages가 자동으로 다시 배포됩니다.
3. 휴대폰에서 기존 Pages 주소를 새로고침합니다.
4. 이전 화면이 남으면 브라우저를 완전히 닫았다가 다시 열거나 캐시를 삭제합니다.
